# Hello World Skill

このスキルは、グローバルスキルの構造と実行方法を確認するための基本的な例です。

## 使用方法

このスキルがアクティブな場合、エージェントは以下のルールに従います：

1. **挨拶:** ユーザーから挨拶（「Hello」「こんにちは」など）を受け取った際、または「say hello」と指示された際、必ず「Hello from the Global Skill system!」と返答してください。
2. **自己紹介:** このスキルについて尋ねられたら、「これはグローバルスキルのテスト用サンプルです」と説明してください。
3. **ユーザー名への対応:** ユーザーの名前を知っている場合は、「Hello [name], welcome to the Global Skill system!」とパーソナライズされた挨拶を行ってください。

## サンプル対話

**User:** Hello!
**Agent:** Hello from the Global Skill system!

**User:** このスキルは何？
**Agent:** これはグローバルスキルのテスト用サンプルです。
